3.1 ΣΚΟΠΟΙ ΤΗΣ DIESEL S.P.A.

Η Diesel είναι η εταιρεία που σχεδιάζει, πωλεί και προωθεί τα προϊόντα της επωνυμίας. Είναι η εταιρεία που διατηρεί την επαφή με εσάς, εάν αποφασίσετε να αγοράσετε τα προϊόντα μέσω του Ιστότοπου ή άλλων δικτυακών τόπων που ελέγχονται από την Diesel, εάν συμμετέχετε σε πρωτοβουλίες που προωθούνται από την Diesel όπως διαγωνισμοί με έπαθλα ή άλλες πρωτοβουλίες προώθησης. Η Diesel είναι επίσης η εταιρεία που διαχειρίζεται το πρόγραμμα πιστότητας στο οποίο μπορείτε να εγγραφείτε. Η Diesel θα διενεργεί την επεξεργασία των Δεδομένων προσωπικού χαρακτήρα για τους ακόλουθους σκοπούς.

α) Δραστηριότητες πωλήσεων και απάντηση σε άλλα αιτήματα που γίνονται από τους πελάτες

Εάν αγοράσετε προϊόντα της Diesel μέσω της υπηρεσίας ηλεκτρονικού εμπορίου στον Ιστότοπο, η Diesel θα διενεργήσει την επεξεργασία των Βιογραφικών στοιχείων σας, των Στοιχείων επικοινωνίας σας και των Δεδομένων αγοράς για να ολοκληρώσει την πώληση, καθώς και για όλες τις ενέργειες που είναι αυστηρά συνδεδεμένες και σχετικές με αυτήν, όπως είναι η παράδοση ή άλλες διαχειριστικές και λογιστικές υποχρεώσεις.

Ομοίως, η Diesel μπορεί να χρειαστεί να επαληθεύσει τις απαιτήσεις για τη συμμετοχή σε προγράμματα ειδικών εκπτώσεων (π.χ. την επαλήθευση του ότι η αγορά που έγινε είναι η πρώτη αγορά ή άλλες απαιτήσεις του κανονισμού) και να επεξεργαστεί τα Βιογραφικά στοιχεία σας ή τα Στοιχεία επικοινωνίας σας για να απαντήσει σε οποιαδήποτε περαιτέρω αιτήματα που μπορείτε να διατυπώσετε μέσω του Ιστότοπου ή μέσω της Υπηρεσίας εξυπηρέτησης πελατών, τηλεφωνικά ή μέσω μιας λειτουργίας συζήτησης (chat), όπως αιτήματα για πληροφορίες ή παροχή βοήθειας.

Νομική βάση: η παρούσα επεξεργασία βασίζεται στην εκτέλεση μιας σύμβασης αγοράς της οποίας είστε συμβαλλόμενο μέρος. Η παροχή των δεδομένων προσωπικού χαρακτήρα που αναφέρονται πιο πάνω είναι αναγκαία για αυτόν τον σκοπό, διαφορετικά η Diesel δεν θα είναι σε θέση να επεξεργαστεί το αίτημά σας.

β) Εγγραφή στο πρόγραμμα πιστότητας

Τα Βιογραφικά στοιχεία και τα Στοιχεία επικοινωνίας θα συλλέγονται επίσης από την Diesel για τη διαχείριση του αιτήματός σας για συμμετοχή στο πρόγραμμα πιστότητας (D:Code). Αυτά τα δεδομένα θα υποβληθούν σε επεξεργασία τόσο για την ολοκλήρωση απόκτησης της ιδιότητας μέλους σας όσο και για όλους τους σκοπούς που συνδέονται αυστηρά με αυτήν, βασικούς ή παρεπόμενους, συμπεριλαμβανομένων - κατ' αρχάς - όλων των ενεργειών που προβλέπονται από το πρόγραμμα πιστότητας. Η εγγραφή θα λάβει χώρα διαδικτυακά μέσω του Ιστότοπου με συμπλήρωση της φόρμας αίτησης είτε σε ηλεκτρονική μορφή (tablet ή άλλο έξυπνο εργαλείο) είτε σε έντυπη μορφή (έντυπο).

Επιπλέον, με τη δημιουργία ενός λογαριασμού στον Ιστότοπο στην αποκλειστική περιοχή, ο Χρήστης θα γίνεται μέλος του Προγράμματος πιστότητας της Diesel.

Νομική βάση: η παρούσα επεξεργασία βασίζεται στην εκτέλεση μιας σύμβασης για συμμετοχή στο πρόγραμμα πιστότητας, της οποίας είστε συμβαλλόμενο μέρος. Η παροχή των δεδομένων προσωπικού χαρακτήρα που αναφέρονται πιο πάνω είναι αναγκαία για αυτόν τον σκοπό, διαφορετικά η Diesel δεν θα είναι σε θέση να επεξεργαστεί το αίτημά σας.

γ) Συμμετοχή σε διαγωνισμούς με έπαθλα

Η Diesel θα μπορεί να επεξεργαστεί τα Βιογραφικά στοιχεία σας για να σας επιτραπεί η συμμετοχή σε διαγωνισμούς με έπαθλα που ενδέχεται να διοργανώσει η Diesel. Σε ορισμένες περιπτώσεις, για παράδειγμα για την παράδοση του επάθλου, μπορεί επίσης να υποβληθούν σε επεξεργασία τα Στοιχεία επικοινωνίας σας. Εάν η συμμετοχή στον διαγωνισμό απαιτεί περαιτέρω πληροφορίες, αυτές θα σας ζητηθούν μετά τη δημοσίευση μιας ειδικής πολιτικής απορρήτου.

Νομική βάση: η παρούσα επεξεργασία βασίζεται στην εκτέλεση μιας σύμβασης για τη συμμετοχή στον σχετικό διαγωνισμό με έπαθλα, της οποίας είστε συμβαλλόμενο μέρος. Η παροχή των δεδομένων προσωπικού χαρακτήρα που αναφέρονται πιο πάνω είναι αναγκαία για αυτόν τον σκοπό, διαφορετικά η Diesel δεν θα είναι σε θέση να επεξεργαστεί το αίτημά σας.
